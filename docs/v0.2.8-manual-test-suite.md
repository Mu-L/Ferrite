# v0.2.8 Manual Test Suite

This document provides a comprehensive manual test checklist for the v0.2.8 release. It covers all new features, bug fixes, and regressions that should be verified before tagging.

**Changelog date:** 2026-04-14

## Pre-Test Setup

- [ ] Build release: `cargo build --release`
- [ ] Have test files ready:
  - Small markdown: `test_md/task_50_table_inline_formatting.md`
  - Large markdown: 6K+ line / 50K+ word document (rendered view perf test)
  - Large text file: 10MB+ (background loading test)
  - 100K+ line file (uniform height mode test)
  - Image files: PNG, JPEG, GIF, WebP, BMP
  - PDF file
  - Files with Arabic, Bengali, Devanagari text (HarfRust shaping)
  - Files with `~~strikethrough~~`, `**bold**` in table cells
  - A Rust or Python file for LSP testing (with `rust-analyzer` or `pyright` installed)
  - Mermaid flowchart test: `test_md/test_flowcharts.md`

## 7. LSP Integration (Phases 1 & 2)

**Prerequisite:** Install `rust-analyzer` (for .rs files) or `pyright` / `pylsp` (for .py files). Enable LSP in Settings → Editor.

### 7.1 Server Lifecycle (Phase 1)
- [ ] Open a .rs file — rust-analyzer starts automatically
- [ ] Status bar shows LSP status (ready / indexing / not found / disabled)
- [ ] Close all .rs tabs — server shuts down after ~30s idle
- [ ] Reopen .rs tab — server re-spawns
- [ ] LSP binary not found — dismissible notification, no crash

### 7.2 Server Crash Recovery
- [ ] Kill the LSP server process externally
- [ ] Ferrite detects crash and restarts with backoff (1s→30s)
- [ ] After 60s uptime, backoff resets

### 7.3 Inline Diagnostics (Phase 2)
- [ ] Introduce a syntax error in .rs file — red squiggly underline appears
- [ ] Introduce a warning — yellow squiggly underline
- [ ] Hover over squiggly — tooltip shows diagnostic message
- [ ] Fix the error — squiggly disappears
- [ ] Diagnostic count shown in status bar

### 7.4 Document Sync
- [ ] Type in editor — changes sent to LSP (didChange)
- [ ] Open new file — didOpen sent
- [ ] Close tab — didClose sent

### 7.5 Settings
- [ ] LSP toggle in Settings → Editor works (opt-in, off by default)
- [ ] Per-language server path overrides configurable
- [ ] **Windows:** No visible cmd.exe window when LSP server spawns
